Tutorials
==============

Tutorials are learning-oriented lessons that guide you through a complete,
hands-on experience with Krum.

Each tutorial takes you from a known starting point to a meaningful result, with every step tested and reproducible.

Available tutorials
-------------------

Getting started
~~~~~~~~~~~~~~~

Your first end-to-end Krum experiment. You will train a small convolutional network on MNIST with a baseline aggregator, then repeat the experiment under a Byzantine attack defended by the Krum rule, and finally compare both runs visually.

This is the best place to start if you are new to the framework.

Benchmarking aggregators
~~~~~~~~~~~~~~~~~~~~~~~~

Compare several robust aggregation rules under the same Byzantine attack.
You will run a small benchmark suite, plot accuracy curves, and learn how
to evaluate rules systematically rather than relying on a single experiment.
